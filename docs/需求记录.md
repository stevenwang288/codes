1) 增加“鼠标选中自动复制”功能（体验参考 OpenCode）。
2) 增加/完善消息通知功能。
3) 调研 OpenCode 原码实现方式并给出可落地方案。

5) CODES 完全独立于 CODEX：
   - 默认数据/配置目录为 `~/.codes`（Windows 示例：`C:\\Users\\<you>\\.codes`），不与 `~/.codex` 混用。

   - 子代理运行过程增加更友好的“进度/状态”展示（类似下载进度条/阶段条），参考现有 `Ctrl+A` 的表达方式并探索更直观的 UI 呈现。
